# Anki Deck Generator для 1С:НИК

Генератор колод Anki для подготовки к экзаменам системы 1С:НИК. Система интервального повторения (SRS) - один из самых эффективных методов запоминания, и этот инструмент использует её для создания идеальной колоды Anki на основе выгрузки данных из приложения "ru.publishing1c.nik".

## Что это делает

Этот скрипт создаёт файлы колоды Anki (.apkg) на основе выгрузки вопросов из приложения 1С:НИК. Получившаяся колода готова к использованию для подготовки к экзаменам с применением мощной системы интервального повторения, которая автоматически определяет оптимальные сроки просмотра карточек.

## Требования

- Python 3.8+
- [Anki](https://apps.ankiweb.net/) (для импорта результата)
- Приложение "1С:НИК" с загруженными вопросами

## Использование скрипта

### Параметры командной строки:

| Параметр | Тип | Обязателен | Описание |
|----------|-----|------------|----------|
| `-f`, `--file` | путь к файлу | Да | JSON файл выгрузки вопросов |
| `-d`, `--data` | путь к директории | Да | Директория с медифайлами (из приложения) |
| `-r`, `--result` | имя файла | Да | Результат колоды (.apkg) |
| `-c`, `--css` | путь к CSS файлу | Нет | Опциональный CSS файл оформления |

### Примеры вызова:

```bash
# Базовый вызов без кастомных стилей
python anki_deck_generator.py -f questions.json -d /path/to/data/ -r result.apkg

# С пользовательским оформлением
python anki_deck_generator.py -f questions.json -d /path/to/data/ -r styled_result.apkg -c styles.css
```

## Инструкция по применению

### 1. Подготовка приложения "1С:НИК"

#### Установка и настройка:

1. Скачайте приложение "1CNIK" для Android/iOS
2. Подпишитесь на нужный экзамен в приложении
3. Загрузите вопросы на устройство (требуются права root)

#### Выгрузка данных:

1. Перенесите папку приложения из `/data/data/ru.publishing1c.nik` на компьютер
   - Включите отладку USB для передачи файлов
2. Откройте приложение в конфигураторе 1С
3. Создайте роль с полными правами
4. Запустите обработку "ЭкспортДанныхJSONИзКонфигурации1НИК.epf"
5. Выберите экзамен и сохраните JSON файл

### 2. Генерация колоды

```bash
# Пример:
python anki_deck_generator.py \
    -f /path/to/questions.json \
    -d /path/to/data/ \
    -r nik_exam.apkg \
    -c styles.css
```

### 3. Импорт в Anki

1. Скачайте [Anki](https://apps.ankiweb.net/)
2. Откройте программу
3. Файл → Импортировать
4. Выберите файл `.apkg`
5. Нажмите "Импорт"
6. Колода появится в списке колод

## Структура колоды

Колоды создаются по иерархии:
- Экзамен (корневая колода)

## Устранение ошибок

**Ошибка: Не найдены медиафайлы**  
Решение: Проверьте путь `-d` должен указывать на корень директории с медифайлами.

**Ошибка: JSON не загружается**  
Решение: Проверьте кодировку UTF-8 в файле вопросов.
